Comment et pourquoi le code est t-il divisé en plusieurs fichier?

1. Le code est mieux structuré, plus logique et plus lisible, pratique si l'on doit travailler en groupe 
2. Pour pas avoir des fichiers trop chargés 
3. Pour voir faire des scripts sur des fichiers et des bouts de code spécifiques 

Comment le nom du pays est t-il affiché quand la souris passe sur une bulle?

- Quand la souris passe sur une bulle, sa position est calculée par rapport à l'objet SVG. Le nom du pays, devient le contenu de la bulle. Et l'élément est positionné au dessus de la souris.
